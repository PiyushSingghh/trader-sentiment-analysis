# Trader Sentiment Analysis

## Objective
Exploring the relationship between 
trader performance and market sentiment
to uncover hidden patterns and deliver
smarter trading strategies.

## Datasets
- Historical Trading Data (211,224 trades)
- Bitcoin Fear & Greed Index (2018-2025)

## Key Findings
- Overall win rate: 83.2%
- Best sentiment to trade: Extreme Greed 
  ($130.21 avg profit, 89.2% win rate)
- Worst sentiment: Extreme Fear 
  ($71.03 avg profit, 76.2% win rate)
- Most traded coin: HYPE

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- XGBoost

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn, XGBoost
- Google Colab
